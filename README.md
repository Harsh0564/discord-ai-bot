# Discord AI Assistant 🤖

A Discord-based AI chatbot that can answer questions, rewrite text, and summarize conversations using Generative AI.

---

## 🚀 Features

- 💬 Ask questions using AI (`/ask`)
- ✍️ Rewrite text professionally (`/rewrite`)
- 📊 Summarize recent chat (`/summarize`)
- 🧠 Context-aware responses using chat history
- ⚡ Fast and interactive Discord slash commands

---

## 🛠 Tech Stack

- Python
- discord.py
- SQLite (for chat memory)
- Gemini API (Generative AI)
- python-dotenv
